## How to use this list

1. **Start in S/A tier** — these have proven buyer intent. The AI-image lane (headshot, eraser, restore, interior) converts hardest and is cheapest to differentiate by re-niching.
2. **One angle per app (Guideline 4.3):** never reskin. Re-*niche* — 'AI headshots for nurses', 'receipt scanner for Uber drivers'. Distinct function, screenshots, keywords.
3. **Spread risk across categories** — don't ship ten habit trackers. The category spread in `niches.json` is intentional.
4. **C-tier = pipeline practice**, not revenue. Use one to learn the ship process end-to-end, then move up.
5. **AI niches** carry per-call cost (FAL/Replicate/OpenAI). Price the paywall so a subscriber's usage stays profitable; cap free generations hard.

Feed any `id` from this list into the generator: `python3 generate.py <id>`.
